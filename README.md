### What I Build

I build platforms that turn data and large language models into reliable products people actually use.

- **AI / LLM systems:** multi-provider gateways, agents, tool calling, RAG, model serving, request telemetry, token-level cost metering, grounded generation and production guardrails.
- **Data platforms:** lakehouse architectures, Spark pipelines, Snowflake, Iceberg, Delta Lake, BigQuery, Databricks, CDC ingestion, orchestration, cataloging, lineage, data quality and governance.
- **Cloud infrastructure:** AWS-first platform engineering with strong multi-cloud experience across Azure and GCP; Terraform/CDK/CloudFormation, containers, serverless, CI/CD and observability.
